Perguntas Frequentes (FAQ)

Tire suas dúvidas sobre o GrowMind e nossa plataforma.

Dúvidas Comuns

O que é o GrowMind?

O GrowMind é uma plataforma de mini cursos corporativos voltada para empresas que desejam aprimorar a integração (onboarding), o bem-estar e o desenvolvimento de seus colaboradores.

Para quem é esta plataforma?

Nossa solução é vendida (modelo B2B) para empresas de qualquer porte que estejam contratando novos funcionários ou queiram reciclar o conhecimento dos times atuais em habilidades humanas e sociais.

Quais cursos são oferecidos?

Atualmente, focamos em três pilares essenciais:

  • Educação Psicológica (autoconsciência, empatia)
  • Educação Financeira (gestão de salário, metas)
  • Desenvolvimento Social (diversidade, inclusão)
Como funciona o progresso?

Cada curso é composto por um vídeo curto e um quiz rápido. Ao completar o quiz, o módulo é marcado como concluído e o funcionário pode avançar para a próxima trilha.

Este projeto usa um banco de dados?

Não. Para fins deste projeto acadêmico, o GrowMind é um protótipo 100% Front-End (HTML, CSS, JS). O progresso e a validação são simulados no navegador usando JavaScript, sem um backend real.
